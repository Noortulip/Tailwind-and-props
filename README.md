# Tailwind and Props

A React project built to practice core React concepts including component props and Tailwind CSS styling.

## What I Learned

- Passing data between components using **props**
- Styling components with **Tailwind CSS**
- Building reusable React components

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

Clone the repository and run:

```bash
npm install
npm run dev
```

Then open your browser at `http://localhost:5173`
